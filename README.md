# IA.RA - Lotus Initiative 🪷

**IA.RA** is a technological solution for personal assistance and protection designed to promote female autonomy in urban environments. The project uses Artificial Intelligence to transform the smartphone into a passive safety device, capable of responding to voice commands without the need for physical interaction.

## 🚀 About the Project
Unlike traditional safety apps that require unlocking the phone, IA.RA operates in the background, monitoring specific sound triggers to execute safety protocols immediately and discreetly.

### Key Features
- **Hands-Free Voice Recognition:** Activation via a pre-configured voice command.
- **Protocol Automation:** Automatic real-time location and status updates sent to support networks.
- **Hybrid Architecture:** Mobile interface built with React Native and AI processing powered by Python.

## 🛠️ Tech Stack
- **Mobile:** [React Native](https://reactnative.dev/)
- **Backend/AI:** [Python](https://www.python.org/) (FastAPI, SpeechRecognition)
- **Infrastructure:** GitHub Codespaces & Expo Go

## 📁 Repository Structure
- `/app`: Mobile application source code (React Native).
- `/server`: Python API responsible for audio processing.
- `/docs`: Scientific documentation and project logbooks.
